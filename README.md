# FLASK_TMS
A Transport Management System Project .

# To run this repo
## first initiat Flask variable 
for windows: 
> setx FLASK_APP 'app/__ init__.py'

for MacOs/Linux:
> export FLASK_APP='app/__ init__.py'

## then run the app
> flask run
